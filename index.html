tabbed - Chrome Tabs

About

The project with the very simple name "tabbed" tries to come up with yet another tab interface. Why another one, one might ask. Well, first of all the approach taken here emulates the look and feel of tabs as seen in the Google Chrome browser. Rounded edges, overlapping tabs, shadows, eventually even the mouse position dependent gradient will be implemented. In addition, I'm trying to achieve something which is not available in other toolkits: the tabs should be allowed to be dragged between different bars. It's even possible to establish new containers with a simple drag. In theory, it would be even possible to manage different types of containers on one site, using only one script. Ain't that something?

Demo

A demo of the initial (and current) state of the project can be found here: http://zemion.github.io/tabbed/demo/ Try and create new tabs, drag them out to establish new containers, and drag tabs between them. Many things unfortunately aren't supported yet. But it's a start, isn't it?
